# Tjansestruktur 🏫

**Tjansestruktur** er en moderne, visuel duksetavle designet med et dybt fokus på specialpædagogik, social tryghed og eksekutive funktioner. Applikationen hjælper lærere og pædagoger med at skabe forudsigelige rammer gennem intelligente makkerskaber og en retfærdig fordeling af opgaver.

## ✨ Nye Intelligente Funktioner (v15)

Vi har opgraderet applikationens hjerte for at sikre maksimal tryghed for eleverne:

*   🤖 **Enhed-baseret Tildeling:** Makkerpar behandles som én samlet enhed. Hvis en elev trækkes til en tjans, følger den faste makker altid med automatisk.
*   📅 **Levende Historik:** Systemet arkiverer automatisk forrige uges plan hver mandag (eller manuelt). Algoritmen tjekker nu både individet og makkerens historik for at undgå gengangere.
*   ⚖️ **Prioriteret Bemandings-logik:** Vigtige opgaver (stjernemarkerede) garanteres altid fuld bemanding (2 elever), før almindelige opgaver fordeles – essentielt for små klasser.
*   🚫 **Strenge Udelukkelser:** Definer elever, der aldrig må arbejde sammen. Systemet overholder dette 100% i alle lodtrækninger.
*   🔔 **Rolige Notifikationer:** Alle forstyrrende `alert`-bokse er erstattet af et diskret banner-system (Toasts) i bunden af skærmen.

## 🧠 Pædagogiske Principper

*   🤝 **Social Tryghed via Roller:** Ved at definere faste under-opgaver (Rolle 1 & 2) fjernes behovet for social forhandling, hvilket minimerer konflikter i makkerskabet.
*   🎯 **Succeskriterier (The Power of Done):** Hver tjans har et konkret mål (stoppunkt), så eleven præcis ved, hvornår opgaven er løst med succes.
*   🌀 **Arousal-regulering:** Valg mellem fire rolige bland-animationer (Spinner, Puls, Indfase, Flimmer) for at gøre skiftet til en forudsigelig og faktuel oplysning.
*   🎨 **Visuel Ro:** Mulighed for high-contrast mode, bløde former og dot-pattern struktur for at skabe et fokuseret arbejdsmiljø.

## 🚀 Tekniske Funktioner

*   💾 **Privacy First:** Alle data (klasseliste, parringer, historik) gemmes udelukkende i din egen browsers `localStorage`.
*   📂 **Backup-sikkerhed:** Eksporter hele din opsætning til en JSON-fil som backup eller til brug på andre computere.
*   🖨️ **Print-klar:** Automatisk formatering til A4-print med ugenummer, farver og tydelige navnefelter.

## 🛠️ Installation & Udvikling

For at køre projektet lokalt:

1.  Klon repository:
    ```bash
    git clone https://github.com/DIT_BRUGERNAVN/StrukturTavlen.git
    ```
2.  Installer afhængigheder:
    ```bash
    npm install
    ```
3.  Start udviklingsserver:
    ```bash
    npm run dev
    ```
