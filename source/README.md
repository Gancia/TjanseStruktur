# Tjansestruktur 🏫

**Tjansestruktur** er en moderne, web-baseret duksetavle designet med fokus på visuel støtte, social tryghed og eksekutive funktioner i klasselokalet. Værktøjet er skabt til lærere og pædagoger, der ønsker at skabe et roligt og forudsigeligt læringsmiljø gennem klare makkerskaber og tydelige forventninger.

## ✨ Pædagogiske Funktioner

Applikationen er bygget på et solidt pædagogisk grundlag med fokus på inklusion og specialpædagogik:

*   🤝 **Rolle-definition:** Mulighed for at give hver makker en specifik under-opgave for at mindske social forhandling og konflikter.
*   🎯 **Succeskriterier (The Power of Done):** Definer konkrete mål for opgaverne, så eleverne ved præcis, hvornår de er færdige.
*   🧠 **Arousal-regulering:** Valgfri bland-animation for at undgå stress og unødig spænding i klassen.
*   🎨 **Lav-arousal design:** Vælg mellem beroligende farvetemaer (Natur, Ro, Himmel) for at minimere visuel overstimulering.
*   👁️ **Visuel Støtte:** High-contrast mode og valg mellem bløde eller skarpe visuelle rammer.

## 🚀 Tekniske Funktioner

*   💾 **Total Local Storage:** Ingen brug for servere eller login. Alle data (klasseliste, opgaver, historik) gemmes sikkert lokalt i din browser.
*   📂 **Backup & Import:** Eksporter dine data til en JSON-fil og tag dem med til en anden computer.
*   🎲 **Smart Bland:** Algoritme der tager højde for historik (hvem har sidst haft tjansen) og prioriterede opgaver.
*   🖨️ **Print-optimeret:** Designet til at blive printet på A4 med alle farver, mønstre og ugenummer intakt.

## 🛠️ Installation & Udvikling

For at køre projektet lokalt:

1.  Klon repository:
    ```bash
    git clone https://github.com/DIT_BRUGERNAVN/StrukturTavlen.git
    ```
2.  Installer afhængigheder:
    ```bash
    npm install
    ```
3.  Start udviklingsserver:
    ```bash
    npm run dev
    ```
